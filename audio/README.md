# Audio Folder

Thư mục này dùng để chứa file ghi âm theo tên trong `data/utterances.csv`.

Ví dụ:

```text
audio/vise_0001.wav
audio/vise_0002.wav
```

Khuyến nghị ghi âm:

- Định dạng: WAV, mono.
- Sample rate: 16 kHz hoặc 44.1 kHz.
- Mỗi file chỉ chứa một câu.
- Giữ giọng tự nhiên như đang nói trong họp kỹ thuật.
- Không cần phát âm tiếng Anh quá chuẩn; nên giữ cách nói quen thuộc của lập trình viên Việt Nam.
